# BMI Rechner

Ein einfacher BMI-Rechner mit einer HTML-Oberfläche und einem Node.js-Server.

## Installation
1. Repository klonen:
   ```sh
   git clone <repo-url>
   cd bmi
   ```
2. Abhängigkeiten installieren:
   ```sh
   npm install
   ```

## Nutzung
Server starten:
```sh
npm start
```

## Technologien
- HTML, CSS, JavaScript (Frontend)
- Node.js mit Express (Backend)
